Hello! This project is for AAIB: *Optimal Routing in Hybrid Chatbot–Human Call Centers with Task Heterogeneity*. 

It studies how a call center should optimally route customer tasks between chatbots and heterogeneous human agents, taking into account differences in task type, difficulty, service time, and resolution probability. 
The goal is to understand the trade-offs between First Contact Resolution (FCR), service efficiency, and operational cost under constrained staffing and infrastructure.
